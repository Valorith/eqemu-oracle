# Docs Extensions

Create JSON files with a `pages` array.

```json
{
  "pages": [
    {
      "id": "custom/example-page",
      "path": "custom/example-page",
      "slug": "custom-example-page",
      "title": "Example Page",
      "markdown": "# Example Page\n\nCustom docs content.",
      "aliases": [],
      "tags": ["custom"],
      "mode": "augment"
    }
  ]
}
```
